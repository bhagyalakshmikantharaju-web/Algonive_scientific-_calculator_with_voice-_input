Scientific Calculator with Voice Input

📌 Project Description

Scientific Calculator with Voice Input is a Python-based application that performs both basic and scientific mathematical calculations. The calculator allows users to enter expressions using buttons, keyboard typing, or voice commands.

The system also provides audio feedback for the result using text-to-speech, making the calculator more interactive and user-friendly.

This project is developed using Python libraries such as Tkinter for the graphical user interface, SpeechRecognition for voice input, and pyttsx3 for converting text results into speech.

---

✨ Features

- Basic arithmetic operations (+, −, ×, ÷)
- Scientific calculations (sin, cos, tan, log, etc.)
- Voice input for performing calculations 🎤
- Audio feedback for results 🔊
- User-friendly graphical interface
- Error handling for invalid expressions

---

🛠 Technologies Used

- Python
- Tkinter (GUI)
- SpeechRecognition (Voice input)
- pyttsx3 (Text-to-Speech)

⚙ Installation

1. Clone the repository:

git clone https://github.com/yourusername/Scientific-Calculator-With-Voice-Input.git

2. Navigate to the project folder:

cd Scientific-Calculator-With-Voice-Input

3. Install required libraries:

pip install -r requirements.txt

---

▶️ How to Run

Run the Python file:

python calculator.py

---

🎤 Voice Command Example

Example voice input:
